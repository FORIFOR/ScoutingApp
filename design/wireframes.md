# 「ファンがスカウトマンになれる」スポーツビジネスサービス ワイヤーフレーム設計

## 1. 概要

本ドキュメントでは、「ファンがスカウトマンになれる」スポーツビジネスサービスのiOSアプリケーションの主要画面のワイヤーフレーム設計を定義します。ワイヤーフレームは、ユーザーフロー図に基づいて設計されており、アプリケーションの基本的なレイアウトと機能を視覚化しています。

## 2. 共通UI要素

### 2.1 ナビゲーションバー
```
+----------------------------------+
|  ← 戻る  |  画面タイトル  |  ⋮   |
+----------------------------------+
```

### 2.2 タブバー（ファンユーザー）
```
+----------------------------------+
| ホーム | スケジュール | レポート | プロフィール |
+----------------------------------+
```

### 2.3 タブバー（クラブユーザー）
```
+----------------------------------+
| ホーム | 試合管理 | レポート | テンプレート |
+----------------------------------+
```

### 2.4 アクションボタン
```
+------------------+
|  + アクション    |
+------------------+
```

3. アクセシビリティに配慮し、VoiceOverなどの支援技術との互換性を確保します。

4. 画面サイズやデバイスの違いに対応するため、Auto Layoutを使用した適応的なレイアウトを実装します。

5. ダークモードにも対応し、ユーザーの設定に応じた表示を提供します。
